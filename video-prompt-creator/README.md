# Video Prompt Creator

A plugin designed to assist in creating high-quality, cinematic video prompts for Sora 2.

## Features

- **Video Prompt Architect**: An agent specialized in crafting detailed video prompts.
- **Create Video Prompt**: A skill to generate prompts based on the "Mega Prompt" structure.
- **Generate Storyboard**: A skill to create detailed shot lists and storyboards.

## Usage

1.  Invoke the **Video Prompt Architect** agent.
2.  Ask it to create a prompt for a specific scene or concept.
3.  Ask it to generate a storyboard for a sequence.
